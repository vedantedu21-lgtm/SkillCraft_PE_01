## 📝 Task 1 — Writing Better Prompts

**Objective:**
Learn the basics of prompt engineering by experimenting with clarity, specificity, and tone.

**What I did:**
- Took 3 vague or ineffective prompts
- Rewrote them into improved, specific versions
- Compared and analyzed the outputs to understand how prompt design affects LLM behavior

**Examples covered:**
- Climate Change explanation
- Resume Writing
- Instagram Marketing Ideas

**Key Learning:**
The more specific and structured a prompt is — in terms of audience, format, tone, and context — the more accurate and useful the AI response becomes.
